### Installation

1. Clone the repo

```sh
git clone https://github.com/AgungWicaksonoPutro/FrontEnd-POS-Apps-Icafe.git
```

2. Install packages

```sh
yarn install
```

3. Compiles and hot-reloads for development
```sh
yarn serve
```
